User Authentication & Authorization

User Authentication & Authorization

Overview

The Esoclusty E-Learning Platform implements a comprehensive authentication and authorization system that ensures secure access to platform features while providing a seamless user experience. This section explores the multi-role user management system, JWT-based authentication, and role-based access control.

Multi-Role User Management

User Role System

The platform implements a sophisticated three-tier role system:

Admin Role

  • Complete System Administration: Full access to all platform features
  • User Management: Create, modify, and delete user accounts
  • Role Assignment: Assign and modify user roles and permissions
  • System Monitoring: Access to system logs, metrics, and health status
  • Content Moderation: Oversee and moderate all platform content
  • Database Management: Access to database administration tools
  • System Configuration: Modify platform settings and configurations

Teacher Role

  • Course Management: Create, edit, and delete courses
  • Content Creation: Upload and organize learning materials
  • Student Monitoring: Track student progress and performance
  • Assessment Management: Create and grade quizzes and assignments
  • Communication Tools: Interact with enrolled students
  • Analytics Access: View course-specific analytics and reports
  • Content Publishing: Publish and schedule course content

Student Role

  • Course Enrollment: Browse and enroll in available courses
  • Content Consumption: Access course materials and lessons
  • Progress Tracking: Monitor learning progress and achievements
  • Assessment Participation: Complete quizzes and assignments
  • Communication: Interact with teachers and fellow students
  • Certificate Generation: Receive completion certificates
  • Profile Management: Update personal information and preferences

JWT Token Structure

Token Components

{
  "header": {
    "alg": "HS256",
    "typ": "JWT"
  },
  "payload": {
    "sub": "user@example.com",
    "role": "TEACHER",
    "userId": "507f1f77bcf86cd799439011",
    "iat": 1640995200,
    "exp": 1640998800,
    "iss": "esoclusty-elearning",
    "aud": "elearning-users"
  },
  "signature": "HMACSHA256(base64UrlEncode(header) + '.' + base64UrlEncode(payload), secret)"
}

Token Claims

  • sub (Subject): User email address
  • role: User role (ADMIN, TEACHER, STUDENT)
  • userId: Unique user identifier
  • iat (Issued At): Token creation timestamp
  • exp (Expiration): Token expiration timestamp
  • iss (Issuer): Token issuer identifier
  • aud (Audience): Token audience
Token Management

Token Lifecycle

// Token generation on successful authentication
public String generateToken(UserDetails userDetails) {
    Map<String, Object> claims = new HashMap<>();
    claims.put("role", userDetails.getAuthorities().stream()
        .map(GrantedAuthority::getAuthority)
        .collect(Collectors.joining(",")));
    claims.put("userId", userDetails.getUsername());
    
    return Jwts.builder()
        .setClaims(claims)
        .setSubject(userDetails.getUsername())
        .setIssuedAt(new Date(System.currentTimeMillis()))
        .setExpiration(new Date(System.currentTimeMillis() + JWT_EXPIRATION))
        .signWith(SignatureAlgorithm.HS256, JWT_SECRET)
        .compact();
}

// Token validation on each request
public boolean validateToken(String token, UserDetails userDetails) {
    final String username = extractUsername(token);
    return (username.equals(userDetails.getUsername()) && !isTokenExpired(token));
}

Token Security Features

  • Secure Secret: Cryptographically strong secret key
  • Expiration Time: Configurable token lifetime
  • Signature Verification: HMAC-SHA256 signature validation
  • Claim Validation: Role and user ID verification
  • Token Refresh: Automatic token renewal for active sessions

// Password encoding during registration
public User registerUser(UserRegistrationDTO registrationDTO) {
    User user = new User();
    user.setEmail(registrationDTO.getEmail());
    user.setPassword(passwordEncoder.encode(registrationDTO.getPassword()));
    user.setRole(registrationDTO.getRole());
    return userRepository.save(user);

// Password verification during login
public boolean authenticateUser(String email, String rawPassword) {
    User user = userRepository.findByEmail(email)
        .orElseThrow(() -> new UsernameNotFoundException("User not found"));
    
    return passwordEncoder.matches(rawPassword, user.getPassword());
}

Password Policy

  • Complexity Requirements: Uppercase, lowercase, numbers, special characters
  • Common Password Prevention: Block common weak passwords
  • Password History: Prevent password reuse
  • Expiration Policy: Regular password change requirements
Session Management

Session Security

@Configuration
@EnableWebSecurity
public class SecurityConfig {
    
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .sessionManagement()
                .sessionCreationPolicy(SessionCreationPolicy.STATELESS)
                .and()
            .csrf()
                .disable() // JWT handles CSRF protection
            .authorizeHttpRequests()
                .requestMatchers("/api/auth/**").permitAll()
                .requestMatchers("/api/admin/**").hasRole("ADMIN")
                .requestMatchers("/api/teacher/**").hasRole("TEACHER")
                .requestMatchers("/api/student/**").hasRole("STUDENT")
                .anyRequest().authenticated()
            .addFilterBefore(jwtAuthenticationFilter, UsernamePasswordAuthenticationFilter.class);
        
        return http.build();
    }
}

Session Features

  • Token Expiration: Automatic session termination
  • Concurrent Sessions: Support for multiple device logins
  • Session Invalidation: Secure logout and token blacklisting
  • Remember Me: Extended session for trusted devices

Registration Process

User Registration Flow

@RestController
@RequestMapping("/api/auth")
public class AuthController {
    
    @PostMapping("/register")
    public ResponseEntity<AuthResponseDTO> register(@RequestBody @Valid UserRegistrationDTO registrationDTO) {
        // Validate email uniqueness
        if (userService.existsByEmail(registrationDTO.getEmail())) {
            throw new EmailAlreadyExistsException("Email already registered");
        }
        
        // Create user account
        User user = userService.registerUser(registrationDTO);
        
        // Generate JWT token
        String token = jwtService.generateToken(user);
        
        // Send welcome email
        emailService.sendWelcomeEmail(user.getEmail());
        
        return ResponseEntity.ok(new AuthResponseDTO(token, user.getRole()));
    }
}

Login Process

User Authentication

@PostMapping("/login")
public ResponseEntity<AuthResponseDTO> login(@RequestBody @Valid UserLoginDTO loginDTO) {
    try {
        // Authenticate user
        Authentication authentication = authenticationManager.authenticate(
            new UsernamePasswordAuthenticationToken(loginDTO.getEmail(), loginDTO.getPassword())
        );
        
        // Generate JWT token
        UserDetails userDetails = (UserDetails) authentication.getPrincipal();
        String token = jwtService.generateToken(userDetails);
        
        // Update last login timestamp
        userService.updateLastLogin(loginDTO.getEmail());
        
        // Log successful login
        auditService.logLogin(loginDTO.getEmail(), true, null);
        
        return ResponseEntity.ok(new AuthResponseDTO(token, userDetails.getAuthorities()));
        
    } catch (AuthenticationException e) {
        // Log failed login attempt
        auditService.logLogin(loginDTO.getEmail(), false, e.getMessage());
        throw new InvalidCredentialsException("Invalid email or password");
    }
}

Password Recovery System

Forgot Password Flow

Password Reset Request

@PostMapping("/forgot-password")
public ResponseEntity<Void> forgotPassword(@RequestBody @Valid ForgotPasswordDTO forgotPasswordDTO) {
    String email = forgotPasswordDTO.getEmail();
    
    // Check if user exists
    if (!userService.existsByEmail(email)) {
        // Don't reveal user existence for security
        return ResponseEntity.ok().build();
    }
    
    // Generate reset token
    String resetToken = passwordResetService.generateResetToken(email);
    
    // Send reset email
    emailService.sendPasswordResetEmail(email, resetToken);
    
    return ResponseEntity.ok().build();
}

Password Reset Process

@PostMapping("/reset-password")
public ResponseEntity<Void> resetPassword(@RequestBody @Valid PasswordResetDTO resetDTO) {
    // Validate reset token
    if (!passwordResetService.isValidToken(resetDTO.getToken())) {
        throw new InvalidTokenException("Invalid or expired reset token");
    }
    
    // Reset password
    userService.resetPassword(resetDTO.getToken(), resetDTO.getNewPassword());
    
    // Invalidate reset token
    passwordResetService.invalidateToken(resetDTO.getToken());
    
    // Log password reset
    auditService.logPasswordReset(resetDTO.getEmail());
    
    return ResponseEntity.ok().build();
}

Audit and Logging

Security Event Logging

Audit Service

@Service
public class AuditService {
    
    public void logLogin(String email, boolean success, String failureReason) {
        SecurityAuditEvent event = SecurityAuditEvent.builder()
            .eventType(LoginEventType.LOGIN_ATTEMPT)
            .userEmail(email)
            .timestamp(LocalDateTime.now())
            .ipAddress(getClientIpAddress())
            .userAgent(getUserAgent())
            .success(success)
            .failureReason(failureReason)
            .build();
        
        auditEventRepository.save(event);
    }
    
    public void logRoleChange(String adminEmail, String targetEmail, String oldRole, String newRole) {
        SecurityAuditEvent event = SecurityAuditEvent.builder()
            .eventType(LoginEventType.ROLE_CHANGE)
            .userEmail(adminEmail)
            .targetEmail(targetEmail)
            .oldValue(oldRole)
            .newValue(newRole)
            .timestamp(LocalDateTime.now())
            .build();
        
        auditEventRepository.save(event);
    }
}
